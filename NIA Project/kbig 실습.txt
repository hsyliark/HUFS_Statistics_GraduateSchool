KBiG 분석실습 교육콘텐츠를 사용해 볼 수가 있습니다.

http://www.kbig.kr 접속(회원가입 후) -> 로그인 -> 인프라 -> 교육실습 -> 실습신청 
-> 박재원/(주)솔트룩스/숙대 멀티미디어학과 실습 신청

- 신청하시고 저에게 계정정보를 알려주시면 승인되도록 처리하겠습니다.

승인이후 -> 실습신청관리 -> 서버보기 -> 시작 클릭 -> 서버상태 : 서버가동중 ->
원격데스크톱 연결 -> 원격 접속 정보 입력 -> ID/PW 입력 후 접속
이후 실습과정은 웹 메뉴얼을 따라 사용하시면됩니다.

KBiG 분석실습 교육콘텐츠는 조교들이 직접사용하여 교육하여야 하므로 관심갖고 사용해주시기 바랍니다.

감사합니다.

데이터셋 코드명은 KBiG 실행시 데이터 로드할 때 입력사항에 필요합니다.
